How To Contribute

Before you start

If you are interessted in extending this collection you have to ask yourself some questions:

  • Is your part relevant to this project
  • Is it already part of the collection? Use search to verify this.

  • If you are not shure, may write an E-Mail to

Requirements

  • Github-Account
  • XML-Editor

Contribute

  1. clone repository https://github.com/cobdh/collection
  2. create a separate branch
  3. add a file with your data in data/editors/xxx.xml
  4. add your contribution to data/bibl and data/persons
  5. run formatter and checks
  6. create a pull request to integrate your changes
  7. wait for feedback and solve the requested changes
